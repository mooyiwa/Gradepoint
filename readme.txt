Gradepoint - School Management Software

Pre-requisite - Webserver e.g Wampserver or XAMPP

1. Install Wampserver
2. Create a database named 'gp_offline' from http://localhost/phpmyadmin
3. Import the gp_offline.sql file into the database
4. Create a folder, named gp_offline, in the document root (www in the case of Wampserver)
5. Download and dump the project files into the folder created
6. Launch http://localhost/gp_offline
